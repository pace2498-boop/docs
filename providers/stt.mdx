# STT Providers

The standard pipeline supports multiple Speech-to-Text providers. Set `stt_config.provider` to any of the values below.

---

## Default (AssemblyAI)

When no `stt_config` is provided, AssemblyAI is used with the `universal-streaming-multilingual` model, which supports over 99 languages automatically.

---

## Supported Providers

### AssemblyAI

| Field | Value |
|---|---|
| `provider` | `"assemblyai"` |
| Required Key | `ASSEMBLYAI_API_KEY` |

```json
{
  "stt_config": {
    "provider": "assemblyai",
    "config": {
      "model": "universal-streaming-multilingual",
      "end_of_turn_confidence_threshold": 0.6
    }
  }
}
```

---

### Deepgram

| Field | Value |
|---|---|
| `provider` | `"deepgram"` |
| Required Key | `DEEPGRAM_API_KEY` |

```json
{
  "stt_config": {
    "provider": "deepgram",
    "config": {
      "model": "nova-2-phonecall",
      "language": "en"
    }
  }
}
```

| Field | Value |
|---|---|
| `provider` | `"google"` |

## Provider Comparison

| Provider | Real-time Streaming | Multi-language | Latency | Best For |
|---|---|---|---|---|
| AssemblyAI | ✅ | ✅ 99+ langs | Low | General use, multilingual |
| Deepgram | ✅ | ✅ | Very Low | Phone calls, speed |
| OpenAI | ✅ | ✅ | Medium | OpenAI-first stacks |
| ElevenLabs | ✅ | Limited | Medium | ElevenLabs TTS parity |
| Google | ✅ | ✅ | Low | Google ecosystem |
| Groq | ✅ | ✅ | Very Low | Speed-critical workloads |
